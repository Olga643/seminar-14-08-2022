### Создание фиксации
Для создания фиксации используется команда *git commit*. Для этого в терминале папки с репозиторием необходимо писать команду *git commit -m"Сообщение к коммиту"*. Сообщение к коммиту писать **ОБЯЗАТЕЛЬНО**.
